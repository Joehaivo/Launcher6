## Launcher3 独立可运行版本

> 本仓库按照现代化安卓开发模式, 将AOSP源码中的 `/package/QuickStepLauncher3` 整理成独立可运行的工程, 方便二次开发, 直接用Android Studio打开即可编译运行

> 本仓库包含两个分支版本

| Android-10 (Android Q API-29)          | Android-12 (Android S API-31)          |
